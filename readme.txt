A Delicious Call Out Bar For Your Visitors to Devour

== Description ==

With Lemonade Stand Schedulable Call Out Bar, you can easily show bars on specific dates for things such as holidays and sales. Non-designers can easily design with easy to use color sliders. Also, included is 100+ high quality clipart to make designing the perfect call out bar a breeze!

== Installation ==

1. Install easily with the WordPress plugin control panel or manually download the plugin and upload the folder to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Click "Lemonade Stand" in the dashboard
